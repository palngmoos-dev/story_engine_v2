# 🛡️ 포렌식 제작 로직 (Production Logic)

본 로직은 견본 4.25 양식을 1:1로 완벽하게 미러링하기 위한 기술적 절대 지침이다.

1. **양식 계승**: 새로운 객체를 생성하지 않고, 견본의 기존 셀(Cell)과 텍스트 상자를 타격하여 데이터만 치환한다.
2. **이미지 매칭**: `data_to_photo_truth` 매핑 테이블을 사용하여, 검색 결과가 아닌 검증된 실사 파일만을 매립한다.
3. **중첩 차단**: 텍스트 상자의 경계(Bounding Box)를 계산하여 0.1픽셀의 겹침도 허용하지 않는다.
4. **최종 검수**: 제작 완료 후 `thousand_check_audit.py`를 통해 100억 번의 정밀 감사를 수행한다.
