# 🌐 GitHub 기반 글로벌 옵시디언 마스터 리소스 분석

본 문서는 전 세계 옵시디언 개발자 및 데이터 전문가들이 가장 신뢰하는 기술 리소스를 분석한 결과입니다.

## 1. 전 세계 TOP 5 GitHub 리소스

### [1] Dataview (by BlacksmithGu)
- **핵심**: 마크다운 노트를 데이터베이스처럼 쿼리할 수 있는 기능.
- **적용**: `20_Resources/Story_Assets`에 있는 모든 장면의 메타데이터(장소, 등장인물, 시간)를 자동으로 수집하여 대시보드화함.

### [2] Templater (by SilentVoid13)
- **핵심**: 단순한 텍스트 대체를 넘어 자바스크립트를 실행하는 템플릿 엔진.
- **적용**: 하네스 실행 시 필요한 환경 변수나 파일 구조를 자동으로 생성하는 스크립트화.

### [3] Obsidian Projects (by Sverre)
- **핵심**: 칸반, 캘린더, 테이블 뷰를 통한 프로젝트 관리.
- **적용**: 스토리 전개 과정을 '기획-집필-검토-완료' 단계별로 시각화하여 관리.

### [4] Copilot for Obsidian (by logancyang)
- **핵심**: GPT-4 등 LLM을 옵시디언 내부에 직접 통합.
- **적용**: 현재 집필 중인 내용에 대해 AI와 실시간으로 토론하고 세계관 설정을 검증.

### [5] Smart Connections (by brianpetro)
- **핵심**: 벡터 검색을 통한 노트 간의 지능적 연결.
- **적용**: 수천 개의 파편화된 아이디어 중에서 현재 작업과 가장 연관된 설정을 AI가 능동적으로 추천.

## 🚀 2단계 구현 핵심 전략

1. **지능형 대시보드 (`Dashboard.md`)**: Dataview를 활용하여 `00_Projects`와 `20_Resources`의 상태를 실시간으로 모니터링.
2. **AI 에이전트 인덱싱**: Copilot과 Smart Connections를 설정하여 보관함 전체를 AI의 장기 기억으로 변환.
3. **자동화 워크플로우**: Templater를 사용하여 하네스 명세서 생성을 자동화.
