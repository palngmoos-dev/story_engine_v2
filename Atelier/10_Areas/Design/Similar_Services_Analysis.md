# 🗺️ "시나리오 카드 놀이" 유사 서비스 분석 보고서

본 보고서는 우리 시스템의 독창적인 '카드 기반 서사 엔진'을 고도화하기 위해 전 세계의 유사 서비스를 분석한 결과입니다.

## 1. 글로벌 리서치 결과

### [1] Plottr (Visual Storytelling)
- **특징**: 카드를 타임라인 위에 배치하여 서사 구조를 설계하는 전문 도구.
- **적용**: 옵시디언 칸반(Kanban) 또는 캔버스(Canvas)를 활용하여 '카드 배치형' UI 구축.

### [2] Sudowrite (AI Writer's Companion)
- **특징**: "Rewrite", "Describe", "Expand" 등 작가에게 필요한 AI 기능을 모듈화.
- **적용**: 하네스 명령어로 "현재 카드 확장", "분위기 변경" 등의 기능을 추가.

### [3] Character.ai (Persona Interaction)
- **특징**: 특정 캐릭터의 성격을 가진 AI와 대화.
- **적용**: `20_Resources`의 캐릭터 노트를 기반으로 한 '캐릭터 채팅' 모드 구현.

## 2. 국내 리서치 결과

### [1] 뤼튼 (Wrtn) & 클로바X
- **특징**: 한국어 서사 및 문체에 최적화된 LLM 서비스.
- **적용**: 한국어 정서와 문학적 표현이 중요한 장면 생성 시 참고할 프롬프트 데이터 확보.

## 3. Grand Atelier만의 차별점 (Unique Value)
- **지식 통합**: 단순한 집필 도구가 아닌, PARA 구조의 전체 지능망과 결합됨.
- **실행 자동화**: 하네스 엔진을 통해 복잡한 작업을 명세서 기반으로 정교하게 수행.
- **완전 소유**: 클라우드 서비스에 종속되지 않고 로컬/Git/Drive 3계층으로 내 데이터를 완벽히 소유.
