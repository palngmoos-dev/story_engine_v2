## 📐 The 3-Layer Integration

### 1. Local Workshop (E:\myhub\writing)
- **Engine**: `scripts/execute_task.py` (Harness Engine)
- **Storage**: SSD-speed local storage for AI operations and file management.
- **Access**: Primary interface via **Obsidian**.

### 2. Cloud Bridge (Google Drive)
- **Sync**: Mirrored via `Google Drive for Desktop`.
- **Accessibility**: Real-time sync to mobile/tablet.
- **Workflow**: Capture ideas on the go, AI processes them at home.

### 3. Historical Vault (GitHub)
- **Version Control**: Every design change (ADR) and spec is tracked.
- **Backup**: Remote off-site redundancy.
- **Link**: [palngmoos-dev/mybot_clean](https://github.com/palngmoos-dev/mybot_clean.git)
