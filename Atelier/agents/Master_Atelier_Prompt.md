# Master Atelier Production Prompt (V5.0)

## 1. 제작 철학 (Philosophy)
- **100.0% 싱크로율**: 견본의 모든 디자인 요소(폰트, 크기, 정렬, 색상, 효과)를 픽셀 단위로 복제한다.
- **상세함의 극치**: 짧은 요약은 금지한다. 견본의 텍스트 양에 맞춰 상세한 역사, 문화, 현장 팁을 풍부하게 작성한다.
- **실시간 무결성**: 2026년 공식 홈페이지와 구글 맵의 최신 데이터를 기반으로 시간, 가격, 경로를 1분/1원 단위로 검증한다.

## 2. 체험형 네비게이션 지침 (Experiential Guidance)
- **1분 1초의 미학**: 이동 수단 탑승 시 "어디서, 몇 번을, 어떻게 결제하고, 어디서 내리는지"를 사용자가 현장에서 직접 체험하듯 상세히 기술한다.
- **시각적 이정표**: "우측에 OO 건물이 보이면 좌회전하세요"와 같은 로드뷰 기반의 지각적 정보를 포함한다.
- **실무 데이터**: 2026년 최신 교통 수단(버스, 기차, 셔틀)의 배차 간격, 플랫폼 번호, 요금 결제 방식(트래블월렛, 컨택트리스 등)을 전수 조사하여 기입한다.

## 2. 세부 제작 규칙 (Detailed Rules)
- **텍스트 분량**: 슬라이드 내 가용 공간을 최대한 활용하여, 견본과 동일한 줄 수와 여백을 유지한다.
- **구글 맵/로드뷰 통합**: 모든 이동 경로는 구글 맵의 실시간 추천 경로를 텍스트와 이미지로 상세히 기술하며, 로드뷰의 현장 앵글을 가이드 화살표와 일치시킨다.
- **일정 상세화**: "몇 시 출발, 몇 시 도착, 어디서 주차, 어느 방향으로 진입" 등 현장에서 바로 사용 가능한 수준으로 일정을 쪼개어 기술한다.

## 3. 기술적 강령 (Technical Mandate)
- `pptx_forensic_engine_v44`를 기반으로 모든 객체의 스타일 속성을 선복제 후 데이터 주입한다.
- 이미지 삽입 시 원본의 종횡비를 보존하며, 프레임 내에 시각적으로 가장 완벽한 구도로 피팅한다.
- 모든 결과물은 `output_space`에 타임스탬프와 함께 보관한다.
