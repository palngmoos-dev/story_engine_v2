# 프로젝트: 파국 루트 테스트

## 시스템 철학
이 시스템은 카드 기반 누적형 창작 엔진이다.
- AI는 방향과 추천을 만든다.
- 사용자는 선택과 강제 개입으로 흐름을 바꾼다.
- 감정 / 관계 / 목표 / 기억 / 루트가 동시 누적된다.
- 장면은 이전 선택의 결과를 먹고 자란다.
- 엔딩은 감이 아니라 규칙과 점수로 판정한다.
- 최종 출력은 장면, 추천, 상태, 엔딩, 리포트, Markdown, JSON 저장까지 이어진다.

## 상태
- 루트: collapse_route
- 장면 수: 6
- 평균 목표 달성도: 0.0
